# **App Name**: Samiksha Solutions

## Core Features:

- Homepage Showcase: Display key services, experience, and contact information prominently.
- Service Listings: Categorized list of Civil, Fabrication, and Supply services offered.
- Project Gallery: Display a gallery of completed projects using a masonry layout, complete with lightboxes.
- Vendor Status Verifier: Let site visitors verify Samiksha Enterprises' status as a Registered Vendor using provided credentials.
- Contact Form: Allow users to send inquiries about services or projects with required details. This could be configured with an AI tool that auto-populates services mentioned on the site, aiding in the message composition process.

## Style Guidelines:

- Primary color: Navy blue (#191970) for a professional and trustworthy feel.
- Background color: Light grey (#D3D3D3), creating a clean, modern backdrop.
- Accent color: Gold (#FFD700) for highlighting key elements and CTAs, symbolizing construction/safety accents.
- Body and headline font: 'Inter', a sans-serif for a clean and modern corporate style.
- Note: currently only Google Fonts are supported.
- Use minimalist icons representing each service category to improve visual clarity.
- Implement a responsive grid layout, ensuring the website looks professional on all devices.
- Apply GSAP with ScrollTrigger to create subtle fade-in effects and slide-in animations as the user scrolls, enhancing user experience.
